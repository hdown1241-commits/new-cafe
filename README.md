# new-cafe

A cafe ordering and management web project.

This repository currently includes the project blueprint that defines the planned page structure, file layout, user roles, and implementation tasks.

## Documents

- [BLUEPRINT.md](./BLUEPRINT.md): project structure, page routes, role-based features, and TODO list

## Planned Features

- Customer home page
- Menu list and menu detail pages
- Basket
- Order history
- My page
- Admin menu management
- Admin order management

## Development Direction

The project will be implemented step by step with HTML, CSS, and JavaScript based on the structure described in `BLUEPRINT.md`.
